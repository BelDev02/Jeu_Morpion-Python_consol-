# 🎮 Jeu du Morpion en Python( Jeu_Morpion_Python.py)

## 📌 Description
Ce projet est une implémentation du jeu du Morpion (Tic-Tac-Toe) en Python, jouable en ligne de commande.  
Il permet à deux joueurs de s’affronter en alternant leurs tours jusqu’à la victoire ou un match nul.

## 🎯 Objectif
Ce projet a été réalisé dans le cadre de mon apprentissage en algorithmique et en programmation Python.  
Il met en pratique :
- la gestion des listes
- les structures conditionnelles
- les boucles
- la gestion des entrées utilisateur
- la logique de détection de victoire

## 🛠️ Technologies utilisées
- Python 3
- Module `os` pour l'effacement de l’écran

## ⚙️ Fonctionnalités
- Affichage dynamique du plateau
- Gestion des tours joueurs (❌ / ⭕)
- Vérification automatique des conditions de victoire
- Détection de match nul
- Interface console avec emojis

## ▶️ Exécution

Cloner le projet puis exécuter :

```bash
python Jeu_Morpion_Python.py
